# webdevHausarbeit

## Von Konstantin Twardzik & Marc Hein

###Installation
```
git clone https://github.com/marchein/webdevHausarbeit.git
cd webdevHausarbeit
npm run installandstart
```
###Build only
```
npm run build
```

###API usage
```
/ - help page
/tracks - get all tracks in json format
/track/:id - get the track with the id of :id

```
